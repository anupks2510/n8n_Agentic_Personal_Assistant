# n8n Agentic Personal Assistant

An intelligent, agentic personal assistant powered by **n8n** workflows + **Streamlit** UI.  
It combines automation (n8n) with an interactive web interface (Streamlit) to handle tasks like information lookup, productivity automation, agentic decision-making, and more.

## ‚ú® Features

- **Agentic AI workflows** in n8n for proactive task handling (e.g., email/calendar integration, research, automation chains)
- **Streamlit frontend** for easy user interaction ‚Äî input queries, view results, trigger workflows
- Webhook / API integration between Streamlit and n8n (trigger n8n executions from UI)
- Modular and extensible ‚Äî add more agents/tools in n8n
- Local run support for development and testing
